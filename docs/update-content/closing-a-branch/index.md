# Closing a Branch

Closing a branch touches multiple departments. 

- Website and Catalog: 
    - Alerts: MarCom handles NPL website and Catalog alerts. Shared Systems is back-up. 
    - Locations: Shared Systems. 
- Bedework: Shared Systems. 
- Google, Bing, Yelp: Shared Systems. 
- LibCal: Shared Systems updates hours for affected locations. Individual library staff update event listings, appointments, and meeting room availability.
