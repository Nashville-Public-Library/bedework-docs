# Closed

## Events Calendar Does not Include Holiday or Building Closures

We no longer list holidays or building closures in our Bedework events calendar. Idea being events calendar reserved for showing what **events** are happening or are not happening, due to late cancelation.
 
The question of whether a **facility** is open on any given day becomes a separate matter we communicate elsewhere with each affected party. For Library website, Library handles closure announcements in notifications and locations sections instead of the events calendar.
 
[https://library.nashville.gov/locations/closures](https://library.nashville.gov/locations/closures)
