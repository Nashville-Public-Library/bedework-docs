# Statistics

Generate monthly statistics on number of events in a given period.

1. Go to the public events calendar.
1. Use the filters to select one topic or series to generate statistics on. For example, "Health and Wellness" or "Southeast"
1. At top of events list, choose Export/Subscribe. Shadowbox opens to show options for exporting data. 
1. Data feed or widget, chose `Feed`.
1. Data format, choose `CSV`.
1. Event Count, set to `200`.
1. Choose date range button. 
1. Select Time frame, use Date Range start date and end date. Format your date has YYYYMMDD. For example, to generate statistics for March, 2026, set start date to `20260301` and end date to `20260401`.
1. Click URL provided below. 
1. New browser window displays results in CSV format. 
1. Use Internet Browser tools to save web page as a .csv file. Change the file name and suffix to .csv. For example, `health-march-2026.csv`.
1. Open .csv file in a spreadsheet application, such as Microsoft Excel. 
1. The total number of events equal the number of rows, minus one for the header row.
